# Getting Started

## Prerequisites
- Python 3.8+
- pip

## Installation
1. Clone the repository:
   ```bash
   git clone <repo-url>
   cd quantum-crypto-cryptography
   ```
2. Install dependencies:
   ```bash
   pip install -r requirements.txt
   ```

## Running the Application
```bash
python src/main.py
```

## Running Jupyter Notebooks
```bash
pip install jupyter
jupyter notebook notebooks/
```
